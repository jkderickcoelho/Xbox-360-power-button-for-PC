# Xbox Style PC Power Button Project

This project lets you use a custom button to:
- Turn your PC on
- Play a startup sound using a DFPlayer Mini

## ⚠️ Important Assembly Note

When making the **2x8 cm board for the button**:

- Establish your **metal peg mounting points first**
- Cut the board to fit within the **2 inch diameter hole**
- Install metal mounting pegs by heating them with a soldering iron while inserting them into the plastic  
  *(I recommend watching a quick tutorial on this if you haven’t done it before)*
- Solder your button in place so it lines up with the **plastic button**  
  *(which should already be permanently mounted by melting the ABS pegs on the Xbox 360 button into the housing)*
- Then solder the board down to the metal pegs, which should now be embedded into the plastic holes in the oval standoff

⚠️ Once the board is mounted, it is permanent  
If not done correctly, you will NOT be able to fit it through the 2 inch hole afterward
